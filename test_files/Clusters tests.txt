		Notas ----------------------------------------------------------------------------
Existen 5 estados para pruebas;
	1 es "Pasada"(La prueba termino tal como se tenia previsto),
	2 es "No Pasada"(La prueba no termino tal como se tenia previsto), 
	3 es "Pasada (Sin problemas de ejecución)"(El proyecto pudo manejar un error de prueba de tal manera que no genero problemas de ejecución en el proyecto y/o servidor pero de manera inesperada, puede que el mensaje de error sea diferente al esperado por no tenerlo previsto debido a esto, se cuenta como "pasada" al probarse en un error y generar un error) 
	4 es "No Pasada (Error ejecución)" (El proyecto no pudo manejar un error de prueba de tal manera que genero problemas de ejecución en el proyecto y/o servidor)
